# Cours de Français Avancé — B2 · C1 · C2

Cours interactif autonome pour apprendre le français de niveau avancé : grammaire, expressions natives, phonétique, exercices de production, jeux.

## 🌐 Voir le cours en ligne

👉 **[Ouvrir le cours](https://ldws123.github.io/cours-francais/)**

## 📚 Contenu

- **18 onglets** : modules B2/C1/C2, trucs de natifs, atelier oral, dictée, descriptions d'images, phonétique, proverbes, discours indirect, hypothèses, traduction FR↔EN, faux amis, vrai/faux, jeux, création
- **42 exercices interactifs** avec correction automatique
- **199 questions** à choix multiples
- **406 expressions idiomatiques** classées par 19 thèmes
- **5 jeux** : Memory, Pendu, Quiz chronométré, Construction de phrases, Flashcards
- **5 scènes SVG** à décrire (entraînement DALF C1/C2)
- **4 dictées** avec text-to-speech et correction colorée mot à mot
- **Atelier oral** avec micro (Web Speech API)

## 🛠️ Technologie

- HTML / CSS / JavaScript vanilla — un seul fichier autonome
- Aucune dépendance, aucune requête réseau
- Tous les progrès sauvegardés en local (`localStorage`)
- Fonctionne hors-ligne une fois la page chargée

## 🚀 Utilisation locale

Ouvre simplement `index.html` dans Chrome, Edge ou Firefox.

## 🔒 Vie privée

Aucune donnée envoyée à l'extérieur. Le micro et la synthèse vocale sont traités entièrement par ton navigateur.
